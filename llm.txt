# jscad-ai-studio

AI-assisted 3D modeling with Claude Code using jscad-fluent + jscadui viewer.

## jscad-work Command

```bash
jscad-work <model.js>    # Create/work on model (starts viewer + browser)
jscad-work start         # Start viewer only
jscad-work               # Show help
```

## Workflow

1. Run `jscad-work model.js` to set up directory
2. Write model code
3. Use MCP browser tools to verify:
   - `browser_navigate` to load model
   - `browser_console_messages` to check errors
   - `browser_take_screenshot` to verify visually
4. Iterate based on feedback

## Model Format

```javascript
const jf = require('@jbroll/jscad-fluent');

const main = (p) => {
  p._type = 'Model Name';
  p.size = { type: 'slider', default: 10, min: 5, max: 20, label: 'Size', live: true };

  return jf.cube({ size: p.size }).colorize([0.3, 0.6, 0.8]);
};

module.exports = { main };
```

## Viewer URL

```
http://127.0.0.1:5120#<directory>/<model>.js
```

Hot-reloads on file save.

## Parametric System (jscadui)

Inline parameter definitions via proxy on `p` object:

```javascript
p.count = 5              // int
p.scale = 1.5            // number
p.enabled = true         // checkbox
p.name = 'foo'           // text

p.radius = {
  type: 'slider',        // slider|int|number|color|choice|checkbox|text
  default: 5,
  min: 1, max: 20,
  step: 0.5,
  label: 'Radius',
  live: true
}
```

Special: `p._type = 'Section'` for UI labels.
