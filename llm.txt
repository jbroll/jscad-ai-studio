# jscad-ai-studio

AI-assisted 3D modeling with jscad-fluent + jscadui viewer.

## Usage

```bash
jscad-work my-model.js
```

Creates model, starts viewer, opens browser.

## Viewer URL

```
http://127.0.0.1:5120#<directory>/<model>.js
```

## Model Format

```javascript
const jf = require('@jbroll/jscad-fluent');

const main = (p) => {
  p._type = 'Model Name';
  p.size = { type: 'slider', default: 10, min: 5, max: 20, label: 'Size', live: true };

  return jf.cube({ size: p.size }).colorize([0.3, 0.6, 0.8]);
};

module.exports = { main };
```
